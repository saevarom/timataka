# User Guide: New Features Tutorial

This tutorial will help you get started with the new features in the Timataka Results Web Application.

## Browsing Events

1. **Access the Events Page**:
   - Click on the "Events" link in the header navigation
   - Or go to the homepage and click on the "Browse Events" button

2. **View Event Details**:
   - On the Events page, click on any event card to see races within that event
   - Events are sorted with the most recent at the top

3. **Navigate to Race Results**:
   - From the Event Detail page, click on any race to view its results
   - You'll see all contestants for that specific race

## Using Birth Year Search

To find a specific contestant when multiple people have the same name:

1. **Basic Search**:
   - Type the contestant's name in the search box (e.g., "John Smith")
   - Click the search button or press Enter

2. **Search with Birth Year**:
   - Type the name followed by birth year: "John Smith 1985"
   - Or use parentheses: "John Smith (1985)"
   - The search will prioritize exact birth year matches

3. **Reading Results**:
   - Birth years appear in parentheses next to contestant names
   - If no exact birth year match is found, you'll see a message indicating this

## Tips for Searching

- **Using partial names**: You can type just a first or last name
- **Icelandic characters**: The search works with both Icelandic and standardized characters (e.g., "Jón" or "Jon")
- **Name variations**: If unsure of exact spelling, type a shorter portion of the name
- **Birth year only**: You can search using only a birth year (e.g., "1985") to find all contestants born that year

## Viewing Contestant Details

When viewing a contestant's details, you'll now see:

1. A birth year chip under the contestant's name
2. This helps confirm you've found the right person
3. When looking at race results, birth years are displayed next to names in the results table

We hope these new features make it easier to find and follow your favorite runners!
